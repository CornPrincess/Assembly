# 监测点10.3

下面的程序执行后，ax中的数值为多少？

| 内存地址 |     机器码     |    汇编指令    |
| :------: | :------------: | :------------: |
|  1000:0  |    b8 00 00    |    mov ax,0    |
|  1000:3  | 9a 09 00 00 10 | call far ptr s |
|  1000:8  |       40       |     inc ax     |
|  1000:9  |       58       |   s: pop ax    |
|          |                |   add ax,ax    |
|          |                |     pop bx     |
|          |                |   add ax,bx    |

ax中的数值为1010H，注意执行完call far ptr s后，IP先变为8，然后将CS、IP的值分别为1000和8依此压栈，最后再跳转至s继续执行。
